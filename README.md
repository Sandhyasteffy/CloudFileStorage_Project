# Cloud File Storage (Project 1)

## Description
This is a mini cloud project using **AWS S3**.  
It allows you to **upload, store, and securely download files from the cloud**, just like Google Drive.

## Features
- Upload files to AWS S3
- Private storage (secure)
- Generate presigned URLs for secure sharing
- Download files from cloud

## Skills Learned
- Creating S3 buckets in AWS
- Uploading and downloading files from cloud
- Understanding presigned URLs for secure file sharing
- Basics of cloud storage systems

## How to Use
1. Log in to AWS console
2. Open S3 and select your bucket
3. Upload files to the bucket
4. Use presigned URLs to securely share or download files
